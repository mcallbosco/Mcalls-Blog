Adding A MASM32 Build Task to VS:CODE On All Platforms

If you are like me, you are probably a computer science student who is forced to take an assembly class but has no interest in installing Visual Studio and all the crap that comes with just to use it for a few months. Or maybe you just like to use something else than windows. So, I am sharing some Visual Studio Code tasks to make life easier. These are made to use the IRVINE libraries but feel free to modify them however you would like. If any of the files you need to download goes down, I have a backup in the repository containing this document. Here’s a link to it. You can download the Build Task file using THIS LINK.

                I will be referring to THIS article by Ryan Eberhardt (If this link doesn’t work there’s a backup in the repo)

                WINDOWS

                Install MASM32 by downloading the installer from http://www.masm32.com/ and then download the Irvine library from “Assembly Language for x86 Processors, 8/e (asmirvine.com)” (Click on the VS 2019 setup and then you will find the download link). Extract the Irvine.zip onto your C: drive and then rename the irvine folder to have a capital I.

                Once you have done that, follow the instructions in the article about setting up your environment variables. When you are finished, the MASM build task is ready to use!

                LINUX……… and MAC(Probably?)

                Follow the instructions in the article (including adding the script at the end to your bash_profile). The WGET links are down but you can find the files by going to http://www.masm32.com/ and “Assembly Language for x86 Processors, 8/e (asmirvine.com)” (Click on the VS 2019 setup and then you will find the download link) and do the stuff required manually.

                You should be all set to start building after this by using the MASM Linux build task. It should also work on Macs.
